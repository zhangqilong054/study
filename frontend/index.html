⚙️ 配置说明

本平台支持接入任意兼容 OpenAI 接口的 AI 服务。在 backend/.env 文件中配置：

AI_API_KEY=你的API密钥
AI_API_BASE=https://api.openai.com/v1
AI_MODEL=gpt-3.5-turbo

未配置 API Key 时，平台将以演示模式运行，返回示例结果。

🚀 启动方式

cd backend
pip install -r requirements.txt
python app.py

默认在 http://localhost:5000 启动。
